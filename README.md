# 📊 Power BI Dashboard Project

## 📌 Project Overview
This project is a Power BI dashboard created to analyze and visualize data effectively. It helps in understanding key business insights using interactive charts and KPIs.

## 🎯 Objectives
- To analyze the dataset and extract meaningful insights
- To create an interactive dashboard
- To track key performance indicators (KPIs)
- To support data-driven decision making

## 📂 Dataset
- The dataset used in this project contains structured data for analysis
- Includes fields like sales, performance, or other business metrics (edit based on your dataset)

## 📈 Dashboard Features
- KPI Indicators
- Interactive Filters (Slicers)
- Charts and Graphs
- Trend Analysis
- Clean and user-friendly layout

## 🛠️ Tools & Technologies Used
- Power BI
- Data Cleaning & Transformation
- Data Visualization



## 🚀 How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the dashboard using filters and visuals

## 📌 Key Insights
- Sales increased over time
- Top-performing category identified
- Regional performance comparison
